Admin view for live Firestore data.

This reads directly from the Firebase Admin SDK on the server. It is intentionally simple: users, follows, wishlist items, feedback, and requested brands in one place.

Admin sign in

Use the server-side admin password from Vercel. Firebase credentials stay on the backend only.

There is no stored user name yet in Firestore, so this view uses email plus uid.
